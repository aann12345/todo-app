# Задачи — для семьи и работы

Приложение ежедневных задач в духе Todoist: общие семейные списки (покупки, дела по дому) с обновлением в реальном времени и отдельное рабочее пространство. Веб + PWA (устанавливается на телефон).

**Стек:** React + TypeScript + Vite + Tailwind, Supabase (Postgres, Auth, Realtime). Всё на бесплатных тарифах.

## Настройка Supabase (один раз, ~5 минут)

1. Зайдите на [supabase.com](https://supabase.com) → **Start your project** → войдите через GitHub или email (бесплатно).
2. Создайте проект: **New project** → любое имя (например `family-tasks`), придумайте пароль базы (сохраните его), регион — Central EU (Frankfurt). Подождите ~2 минуты, пока проект создастся.
3. Слева откройте **SQL Editor** → **New query** → вставьте всё содержимое файла [`supabase/schema.sql`](supabase/schema.sql) → нажмите **Run**. Должно выполниться без ошибок.
4. **Отключите подтверждение email** (иначе каждому члену семьи придётся кликать по ссылке из письма):
   Authentication → Sign In / Providers → Email → выключите **Confirm email** → Save.
5. Скопируйте ключи: **Project Settings** (шестерёнка) → **API Keys**:
   - `Project URL`
   - `anon public` ключ
6. В корне проекта скопируйте `.env.example` в **`.env.local`** и вставьте свои значения:

   ```
   VITE_SUPABASE_URL=https://ваш-проект.supabase.co
   VITE_SUPABASE_ANON_KEY=eyJ...
   ```

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173, зарегистрируйтесь — личное пространство создастся автоматически.

## Как пользоваться

- **Пространства** — «Личное» создаётся сразу. Кнопкой «+» создайте «Семью» или «Работу».
- **Пригласить семью:** в пространстве откройте «Участники» → «Создать код приглашения» → передайте 6-значный код. Второй человек регистрируется, жмёт «+» → «Вступить по коду».
- **Realtime:** изменения в общих списках появляются у всех мгновенно, без обновления страницы.
- **Повторы:** в задаче укажите срок и повтор («каждую неделю: сб») — при выполнении появится следующее вхождение.
- **Назначение:** в задаче выберите исполнителя — у него она появится в «Мои задачи».

## Установка на телефон (PWA)

После деплоя (или в локальной сети) откройте сайт в браузере телефона:
- **Android/Chrome:** меню ⋮ → «Установить приложение»
- **iPhone/Safari:** «Поделиться» → «На экран „Домой"»

## Деплой (бесплатно, чтобы семья пользовалась откуда угодно)

Проще всего — [Vercel](https://vercel.com): импортируйте репозиторий GitHub, в настройках проекта добавьте те же две переменные окружения (`VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`) и нажмите Deploy.

## Известные ограничения бесплатного тарифа

Supabase Free «усыпляет» проект после ~7 дней без обращений — если приложением не пользоваться неделю, его нужно разбудить кнопкой Restore в дашборде Supabase. При регулярном использовании этого не происходит.
